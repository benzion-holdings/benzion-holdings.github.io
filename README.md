# Benzion Holdings — Website

Landing page for [benzionholdings.com](https://benzionholdings.com). Static site on GitHub Pages with a custom GoDaddy domain.

## Stack

- HTML + CSS, no frameworks
- GitHub Pages hosting
- Custom domain: benzionholdings.com

## Local preview

```bash
python3 -m http.server 8080 --directory /path/to/benzion-holdings
```

Open [http://localhost:8080](http://localhost:8080).

## Deploy

Push to `main` — GitHub Pages auto-deploys.
